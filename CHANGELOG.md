# Changelog

All notable changes to TokenKite will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-02

### Added
- Claude Code usage tracking with automatic log parsing
- Cursor billing integration
- Quota monitoring for Claude subscriptions (Pro/Max)
- Menu bar interface with quick cost overview
- Per-service dashboards with detailed breakdowns
- Settings for refresh intervals and subscription plans
- Estimated and precise quota tracking modes

### Requirements
- macOS 14.0+ (Sonoma)
- Notarized and signed with Developer ID
